# Calculator API

A simple Flask calculator API.

## Features
- Add , Subtract, Multiply, Divide, Exponentiate, and find Reminder of numbers.
- JSON responses

## Endpoints

/add?a=5&b=10
/sub?a=10&b=5
/mul?a=5&b=2
/div?a=10&b=2
/div?a=7&b=0
/exp?a=2&b=2
/rem?a=15&b=7

## Tech Stack
- Python
- Flask
